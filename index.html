Mis Proyectos

NutriLife

NutriLife es una innovadora herramienta que facilita a los nutricionistas el seguimiento de la salud de sus pacientes. Permite calcular el IMC, registrar información vital, y ofrece visualizaciones claras del estado nutricional.

El Café del Negro José

El Café del Negro José es un sitio web dedicado a brindar una experiencia única y acogedora para los amantes del café. Diseñada y desarrollada exclusivamente utilizando tecnologías de frontend. (Mi Primera Página Web)

Fragaria Perfumes

Fragaria Perfumes es una aplicación web full-stack desarrollada para la gestión y visualización de un catálogo exclusivo de perfumería prémium. El objetivo principal del proyecto fue construir una plataforma intuitiva, escalable y visualmente atractiva utilizando las mejores prácticas de la arquitectura MVC (Modelo-Vista-Controlador).

Patitas y Huellas

Este proyecto es una aplicación web desarrollada con Django que permite gestionar información sobre mascotas y sus dueños. Incluye funciones para registrar, editar y eliminar registros, así como navegar entre vistas mediante una barra de navegación
