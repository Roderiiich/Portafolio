Mis Proyectos

NutriLife

NutriLife es una innovadora herramienta que facilita a los nutricionistas el seguimiento de la salud de sus pacientes. Permite calcular el IMC, registrar información vital, y ofrece visualizaciones claras del estado nutricional.

El Café del Negro José

El Café del Negro José es un sitio web dedicado a brindar una experiencia única y acogedora para los amantes del café. Diseñada y desarrollada exclusivamente utilizando tecnologías de frontend. (Mi Primera Página Web)

Fragaria Perfumes

Fragaria Perfumes es una aplicación web full-stack desarrollada para la gestión y visualización de un catálogo exclusivo de perfumería prémium. El objetivo principal del proyecto fue construir una plataforma intuitiva, escalable y visualmente atractiva utilizando las mejores prácticas de la arquitectura MVC (Modelo-Vista-Controlador).

Patitas y Huellas

Este proyecto es una aplicación web desarrollada con Django que permite gestionar información sobre mascotas y sus dueños. Incluye funciones para registrar, editar y eliminar registros, así como navegar entre vistas mediante una barra de navegación

Nube IPVG

Lideré el diseño técnico y financiero de una infraestructura elástica en AWS para erradicar los colapsos de la intranet institucional durante la toma masiva de asignaturas. Coordiné a un equipo especializado para desplegar un clúster de bases de datos de 3 nodos 100% open-source y estructuré un modelo presupuestario que demostró la prevención de $25M CLP en pérdidas operativas, asegurando la continuidad académica de 10.000 estudiantes.

Parkr App

Parkr es una aplicación de parquímetros desarrollada en Android, diseñada para facilitar el proceso de pago y gestión del estacionamiento. Permite a los usuarios recibir feedback sobre el tiempo de estacionamiento. La interfaz intuitiva y amigable garantiza una experiencia de usuario fluida. ¡Simplifica tu experiencia de estacionamiento con Parkr!

Wilson XD

Wilson XD es un sistema operativo web interactivo y multiusuario. El proyecto simula una experiencia de escritorio retro donde cada usuario autenticado cuenta con un entorno aislado y persistente. Destaca por su sistema de gestión de descargas virtuales ("Lares Downloads"), personalización de interfaz (fondos y avatares) y una innovadora simulación de gestión de recursos del sistema en tiempo real. Si el usuario excede el "espacio en disco" asignado, el sistema despliega un nostálgico y funcional 'Pantallazo Azul de la Muerte' (BSOD).

MotorGet

Este proyecto es una aplicación web desarrollada con Python y Django que permite gestionar automóviles de manera eficiente. Incluye funcionalidades para agregar, editar, eliminar y visualizar vehículos, así como organizar la información de forma estructurada mediante una interfaz amigable.

EduFácil

Es una aplicación web diseñada para simplificar la administración de datos escolares mediante una interfaz limpia e intuitiva. El objetivo principal del proyecto es resolver la complejidad de la gestión académica, permitiendo a los administradores e instituciones educativos controlar de manera eficiente el ciclo de vida completo de los registros de estudiantes y materias, asegurando la integridad de los datos en todo momento.

Nutrilife IA Android

El objetivo de NutriLife IA es ayudar a los usuarios a mejorar su salud mediante una herramienta gratuita, accesible y fácil de usar, promoviendo una vida saludable a través de recomendaciones prácticas y personalizadas.

Portafolio Web

Portafolio profesional e iterativo concebido para centralizar y exhibir mi trayectoria como ingeniero en formación. Desarrollado combinando la robustez de Bootstrap 5 con un diseño de UI personalizado en CSS3 y lógica de control en JavaScript. Destaca por su alto rendimiento, fluidez de navegación interna y una óptima adaptación a cualquier tipo de pantalla.
